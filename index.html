🚀 Quick Setup

Get the bot running in under 5 minutes

1

Install Dependencies

pip install python-telegram-bot requests beautifulsoup4
2

Get Bot Token from BotFather

  1. Open Telegram and search @BotFather
  2. Send /newbot
  3. Choose a name and username for your bot
  4. Copy the bot token you receive
3

Get Your Telegram User ID

Message @userinfobot on Telegram and it will reply with your User ID.

4

Configure & Run

# Option A: Environment variables
export BOT_TOKEN="your_bot_token"
export ADMIN_ID="your_telegram_id"
python bot.py

# Option B: On Windows
set BOT_TOKEN=your_bot_token
set ADMIN_ID=your_telegram_id
python bot.py

# Option C: Edit bot.py directly
# Change these lines:
BOT_TOKEN = "your_token_here"
ADMIN_ID = 123456789
✓

Admin Setup (After Bot Starts)

# Set log channel (forward channel msg to @userinfobot to get ID)
/setchannel -1001234567890

# Allow users to use the bot
/adduser 987654321

# View allowed users
/listusers

# Broadcast to all users
/broadcast Hello everyone!

📋 Bot Commands

All available commands and their usage

👤 User Commands

/start Welcome message with action buttons and developer info
/scrape <url> Scrape a single page's source code and send as file
/scrapeall <url> Crawl all pages on the domain and send each as file
/help Display usage instructions and command reference

🔐 Admin Commands

/setchannel <id> Set the private log channel for activity monitoring
/adduser <id> Grant bot access to a specific user
/removeuser <id> Revoke bot access from a user
/listusers List all users with bot access
/broadcast <msg> Send a message to all bot users
